• XSL (Extensible Style Language) ជាភាសាសម្រាប់រចនា xml
    XSL មានសមាសធាតុផ្សំសំខាន់ៗចំនួន ៣ គឺ ៖​

    1. XSLT (XSL Tranformation) ជាភាសាសម្រាប់ថ្លែងឯកសា xml ជាកសា html, text file, ឬ ឯកសា xml ថ្មីដែលមានរចនាសម្ព័ន្ធផ្សេងពីមុន
    2. XSL-FO (Formatting Object) ជាភាសាមួយសម្រាប់កំណត់ទ្រង់ទ្រាយឯកសា
    3. Xpath (XML Path) ជាភាសាសម្រាប់ចូលទៅកាន់ផ្នែកណាមួយនៃឯកសា html

  • XSLT (Browser -> web server, Tranformation, Stru, Instruction)

  • Web servers ជាប្រភេទ software ម្យ៉ាងដែលលើកកំពសើសមតភាពម៉ាស៊ីន computer ឲ្យអាចផ្ទុកនិង ដំណើរការ website បាន
    គុណសម្បាតិតគឺស្រួលក្នុងការ installations
    គុណវិបត្តិគឺមិនទទួលបាន versions ចុងក្រោយនៃការ update
  • Informative Process ដើម្បីថ្ងៃ xml ជា html ត្រូវការឯកសាដែលប្រើជាធាតុចូលចំនួនពីរគឺឯកសា xml និង ឯកសារ xsl ទើប xsl processer អាចធ្វើការថ្លៃងបាន
    XML Input || XSLT Code ---> XSLT Process ---> Result Document
  • XSL Processer ជាប្រភេទកម្មវិធីដែលអាចដំណើរការប្រភេទ xsl
